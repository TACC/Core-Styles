UI states of [`<a>` anchor element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a) as mixins.

| Mixin | Description
| - | -
| `link`         | default state
| `link--disabled`| "disabled" state (e.g. no `href` attribute)
| `link--hover`  | `:hover` state
| `link--active` | `:active` (click) state
